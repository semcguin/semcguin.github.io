Projects

California Public School Construction Spending

This project looks at a never-before-analyzed data set of California public K-12 school construction since 1994. The report reveals spending trends across construction types and racial demographics.

Crime and Food Trucks in San Francisco

Do food trucks correspond with increased noise violations? What are the most dangerous times and places to be in San Francisco? This and more answered here using SQL to access a database of crime, noise violations, and food trucks in San Francisco.

Understanding College Debt

Here I used the API from CollegeScorecard.gov to extract information on student debt and college tuition. I analyzed this data to determine the relationships between student debt, poverty, and tuition costs.

Web Scraping Nutrition Data

Ever wondered how to get the biggest bang for your buck in the produce aisle? Using an API from the USDA Food Composition Database, I wrote a function to search and extract nutrition facts for different fruits and vegetables. I then made figures relating the produces' price, yeild, and nutritional content to determine which are the most valuable. Avocados for the win!

Restructuring and Analyzing Nutrition Data Files

This project restructures price and nutrition information from an excel file in order to determine which fruits and vegetables are the most valuable in terms of price and yeild.
