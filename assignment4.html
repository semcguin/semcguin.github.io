Assignment 4¶

In this assignment, you'll combine the assignment 3 data set with nutrition data from the USDA Food Composition Databases. The CSV file fresh.csv contains the fresh fruits and vegetables data you extracted in assignment 3.

The USDA Food Composition Databases have a documented web API that returns data in JSON format . You need a key in order to use the API. Only 1000 requests are allowed per hour, so it would be a good idea to use caching.

Sign up for an API key here. The key will work with any Data.gov API. You may need the key again later in the quarter, so make sure you save it.

#Function to make a search request
def ndb_search(search, key):
    #Encoding characters to work in url
    search = urllib.quote(search)
    fresh_uri = "https://api.nal.usda.gov/ndb/search/?format=json&q={}&max=25&offset=0&api_key={}".format(search, key)
    request = Request(fresh_uri)
    response = urlopen(request)
    rr = response.read()
    js = json.loads(rr) #Convert from str into json object
    return js['list']['item']

The first thing I realized is that the most relevant search term comes up first. Therefore, most times if we put in the fruit or vegetable name along with the word "raw", the entry we want is within the first three entries, as long as these entries are not branded.

def get_ndb(food):
    for i in range(len(food)):
        if food[i] == "kiwi":
            food[i]+=str("fruit raw")
        food[i]+=str(" raw")
        food[i] = str.replace(food[i], "_", " ")
        search_return = ndb_search(food[i], key)

        #If the first entry returned by the search function is a branded item, a new function must be called to loop past these entries
        if search_return[0]['group'] == "Branded Food Products Database":
            nbdno = branded(search_return)[0]['ndbno']

        else:
            nbdno = search_return[0]['ndbno']
        ndbnos.append(nbdno)
    return ndbnos

#Gets nutrition data
def ndb_report(key, ndbno):
    fresh_uri = "https://api.nal.usda.gov/ndb/V2/reports/?ndbno={}&type=b&format=json&api_key={}".format(ndbno, key)
    request = Request(fresh_uri)
    response = urlopen(request)
    rr = response.read()
    js = json.loads(rr) #Convert from str into json object
    return js['foods'][0]['food']['nutrients']

def format_data(table): 
    x = [table[1]['value'], table[2]['value'], table[1]['name'], table[2]['name']]
    return x

#Here I modify ndb_report() to include ndbnos with the nutrition for Energy in kcals

def ndb_report_mod(key, ndbno):
    fresh_uri = "https://api.nal.usda.gov/ndb/V2/reports/?ndbno={}&type=b&format=json&api_key={}".format(ndbno, key)
    request = Request(fresh_uri)
    response = urlopen(request)
    rr = response.read()
    js = json.loads(rr) #Convert from str into json object
    
    #subsets json output to only include first row of returned nutritional content
    clean = format_data(js['foods'][0]['food']['nutrients'])
    #add in column for ndbno
    clean.insert(0, ndbno)
    return clean
